Inheritance allows us to write more global styles on parent elements and then "fine-tune" the styles for some of the child elements as needed. Such approach makes writing CSS very efficient.

Subheading 2

Another concept which is semantically similar to bioavailability is bioaccessibility. For a contaminant to be bioaccessible, it must first be dissolved in the gut fluid. A fraction of the dissolved concentration then passes through the intestinal membrane to become bioavailable.
